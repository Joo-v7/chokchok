# chokchok
보습감(1.0 ~ 3.0) 등급을 기준으로 화장품을 정리하고, 사용자들이 자신의 피부 타입에 맞는 제품을 쉽게 찾고 구매할 수 있는 온라인 화장품 쇼핑몰.
> 1인 프로젝트
> 
> 개발 기간: 2025.04.01 ~ 2025.05.31
>
> 기술 스택: Spring Boot 3.4.4, MySQL 9.2.0, Nginx, AWS EC2, ThymeLeaf
>
